# 明日方舟：终末地游戏工具

## 项目介绍

这是一个为《明日方舟:终末地》设计的**基质筛选工具**，帮助玩家快速查找和筛选符合条件的武器及其基质搭配。

## 功能特性

- ✨ **快速筛选**：支持按基础属性、附加属性、技能属性进行多条件筛选
- 📊 **可视化展示**：武器信息卡片化展示，包含图片和详细属性
- 🎯 **精准匹配**：智能词条匹配，支持模糊搜索
- 📱 **响应式设计**：适配各种屏幕尺寸
- 🎨 **简约界面**：白色和灰色主色调，专业简洁

## 数据说明

- 武器：仅收录六星武器和五行武器
- 基质：仅收录五星基质
- 数据来源：官方游戏信息

## 使用说明

### 基本操作

1. 打开 `index.html` 文件
2. 自动加载 `weapon_data.xlsx` 中的武器数据
3. 在下拉框中选择筛选条件（基础属性、附加属性、技能属性）
4. 点击"开始筛选"按钮查看结果

### 文件结构

```
Endfield/
├── index.html           # 主页面文件
├── weapon_data.xlsx     # 武器数据文件（Excel格式）
├── images/              # 武器图片文件夹
├── README.md            # 本说明文件
└── pom.xml              # Maven配置文件（Java后端）
```

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript
- **数据处理**：SheetJS（Excel解析库）
- **后端**：Java（可选）

## Excel数据格式

`weapon_data.xlsx` 应包含以下列：

| 列名 | 说明 | 数据类型 |
|------|------|--------|
| 武器名称 | 武器的名称 | 文本 |
| 武器类型 | 武器的类型分类 | 文本 |
| 基础属性 | 基质基础属性词条 | 文本 |
| 附加属性 | 基质附加属性词条 | 文本 |
| 技能属性 | 基质技能属性词条 | 文本 |
| 图片文件名 | 武器对应图片的文件名 | 文本 |

**示例数据：**

| 武器名称 | 武器类型 | 基础属性 | 附加属性 | 技能属性 | 图片文件名    |
|------|-----|---------|---------|---------|----------|
| 熔铸火焰 | 单手剑 | 智识提升 | 攻击提升 | 夜幕 | rzhy.png |
| 黯色火炬 | 单手剑 | 智识提升 | 灼热伤害提升 | 附术 | ashj.png |

**备注：**
- 所有列都是必需的
- 武器类型可自定义（如：六星、五行、特殊、限定等）
- 词条内容应与 `index.html` 中 `STATIC_OPTIONS` 定义的词条完全匹配
- 图片文件应放在 `images/` 文件夹中，确保文件名正确

## 常见问题

### Q: 为什么显示"加载失败"？
A: 请确保 `weapon_data.xlsx` 文件与 `index.html` 在同一目录，且Excel文件格式正确。

### Q: 怎样添加新的武器？
A: 在 `weapon_data.xlsx` 中添加新行，填写完整的武器和基质信息即可。

### Q: 如何自定义词条？
A: 编辑 `index.html` 中的 `STATIC_OPTIONS` 对象，修改对应属性的词条数组。
